# 마이클 지저스

악보 이미지에서 악보를 추출하고, 사용자가 악보 콘티를 만들어서 악보를 볼 수 있는 웹앱입니다.

## 기술 스택

- TypeScript
- Vite
- Firebase App Hosting

## 개발 환경 설정

1. 의존성 설치
```bash
npm install
```

2. 개발 서버 실행
```bash
npm run dev
```

3. 빌드
```bash
npm run build
```

4. Firebase 배포
```bash
npm run deploy
```

## Firebase 설정

`.firebaserc` 파일에서 프로젝트 ID를 설정하세요.
