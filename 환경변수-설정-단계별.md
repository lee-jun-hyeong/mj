# 환경 변수 설정 - 단계별 가이드

## 현재 위치
Firebase Console > Functions 대시보드에 있습니다.

## 환경 변수 설정 방법

### 방법 1: Functions 환경 변수 페이지로 직접 이동

1. **현재 페이지에서 URL 변경**
   - 브라우저 주소창에서 다음 URL로 이동:
   ```
   https://console.firebase.google.com/project/michael-jesus/functions/env
   ```

2. **또는 사이드바에서 이동**
   - 왼쪽 사이드바에서 "Functions" 아래에 "환경 변수" 메뉴가 있을 수 있습니다
   - 또는 Functions 대시보드 상단의 탭에서 "환경 변수" 탭을 찾아보세요

### 방법 2: 함수 상세 페이지에서 설정

1. **함수 클릭**
   - Functions 목록에서 `processOMR` 함수를 클릭

2. **환경 변수 탭**
   - 함수 상세 페이지에서 "환경 변수" 또는 "Configuration" 탭 찾기
   - "환경 변수 추가" 버튼 클릭

3. **환경 변수 입력**
   - **이름**: `OMR_SERVICE_URL`
   - **값**: `https://omr-service-51418627624.asia-northeast3.run.app`
   - **저장** 클릭

## 설정할 값

- **변수명**: `OMR_SERVICE_URL`
- **값**: `https://omr-service-51418627624.asia-northeast3.run.app`

## 확인

설정 후 Functions 로그에서 확인:
```powershell
firebase functions:log --only processOMR
```

로그에 `omrServiceUrl: https://omr-service-...`가 표시되면 성공입니다.
